## 心得体会

我体会到，AI 是一个优秀的“执行者”，但无法替代决策。我们需要先花大量时间讨论和制定设计规范、梳理页面逻辑，然后再让 AI 去绘制，这种工作模式非常高效。AI 帮我们节省了绘制时间，让我们能把更多精力放在业务逻辑的思考上。
