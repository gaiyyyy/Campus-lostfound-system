## 心得体会

使用AI要学会“先规范、后执行”。以前都是边画边改，页面之间经常出现颜色不一致、间距不统一的问题。但这次我们提前把配色方案、页面逻辑等等都定好，再让Figma AI去生成界面，效率大大提高。AI生成的初稿有80%可以直接用。
