# 设计说明

## 配色方案

- 主色：青绿色#00897B

- 辅助色：浅蓝色#42A5F5

- 强调色：橙色#FF9800

## 字体

- 标题：Roboto

- 正文：Roboto

## 核心页面

1. 登录页 - 支持用户名/密码登录

2. 注册页 - 完整的表单验证（用户名、邮箱、手机号、密码）

3. 主页 - 数据统计面板和功能模块入口（用户/管理员权限自适应）
4. 失物列表页 - 卡片展示、搜索过滤、分类筛选、悬停效果
5. 失物详情页 - 完整信息展示，所有者可编辑/更新状态/删除
6. 失物发布页 - 表单验证、支持失物和招领两种类型
7. 个人中心页 - 个人信息、发布管理、统计数据
8. 系统管理页 - 管理员专属，用户管理和内容审核

## 交互说明

- 未登录用户访问任何页面跳转至登录页
- 注册页注册成功后跳转到登录页

- 点击登录按钮后根据角色跳转到用户端或管理员端主页
- ,从主页可进入失物列表、发布页、个人中心，管理员可从主页进入系统管理模块
- 失物招领列表页点击卡片进入详情页，悬停卡片有轻微阴影或放大效果
- 信息发布页点击提交按钮，发布后跳转到列表页
- 个人中心页可管理用户发布内容，有“我发布的失物”和“我发布的招领”列表，每项可编辑、更新状态、删除
